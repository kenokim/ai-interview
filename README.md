# AI Interview

## Feature list

- Interview session (2 modes)
  - chat (text) based
  - voice based

- Interview types
  - culture-fit / behavioral interview
  - tech interview (backend)
  - JD / CV based interview (planned)

- Session report: evaluates your submissions
- Multilingual support: Korean, English
- Admin: manages interview questions

## Repo structure

- `front/`: React + Vite frontend (TypeScript, Tailwind, shadcn-ui)
- `server/`: Python FastAPI API + ETL
- `docs/`: PRDs, design, and research docs

## Tech stack

- Frontend: React, Vite, TypeScript
- Backend: FastAPI
- AI/Agent: LangGraph, LLM, evaluation
- Voice: Gemini TTS
