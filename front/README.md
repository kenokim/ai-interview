# AI Interview Frontend

This is the React + Vite frontend for the AI Interview project.

## Requirements

- Node.js (LTS recommended)

## Install

```sh
cd front
npm install
```

## Run (dev)

```sh
npm run dev
```

## Build

```sh
npm run build
```

## Preview (local)

```sh
npm run preview
```

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
