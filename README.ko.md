# AI Interview

## 기능 목록

- 면접 세션 (2가지 모드)
  - 채팅(텍스트) 기반
  - 음성 기반

- 면접 유형
  - 컬처핏 / 행동면접
  - 기술 면접 (백엔드)
  - JD / CV 기반 면접 (예정)

- 세션 리포트: 답변 평가
- 다국어 지원: 한국어, 영어
- Admin: 면접 질문 관리

## 레포 구조

- `front/`: React + Vite 프론트엔드 (TypeScript, Tailwind, shadcn-ui)
- `server/`: Python FastAPI API + ETL
- `docs/`: PRD 및 설계/리서치 문서

## 기술 스택

- Frontend: React, Vite, TypeScript
- Backend: FastAPI
- AI/Agent: LangGraph, LLM, evaluation
- Voice: Gemini TTS
